Для чего нужны Generics?
	для обеспечения проверки типов во время компиляции.

Примеры Generic классов в Java?
	public class MyGeneric<T1, T2> {
    		private T1 one;
    		private T2 two;
	}

Почему плохо создавать поля так: Object value; нежели T value?
	если указать как Обжект это не очень удобно,
	мы не сможем использовать методы класса, будут доступны только методы обжекта.
